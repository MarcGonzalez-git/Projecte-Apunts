HTML Bàsic

1. Estructura bàsica

Estructura general

Tota pàgina d'HTML, aquesta inclosa, ha de seguir una estructura bàsica, començant per l'etiqueta < !DOCTYPE >, que marca el tipus de document, i continuant amb diferents etiquetes com < html >, < head > i < body >, que ens indiquen el tipus de informació.

2. Etiquetes bàsiques

Per definir el diferent tipus de text que ens podem trobar a aquests documents, es fa ús de diferents etiquetes bàsiques, que són necessàries per definir el tipus de text.
Dintre d'aquest grup trobam diverses etiquetes, com:

  • < title >: ens permet definir el títol mostrat a la pestanya del navegador, en aquest cas, HTML Bàsic.

  • < h1 > a < h6 >:

    Permet

    Definir

    El

    Tamany

    Dels
    Títols

  • < p >: s'utilitza per definir blocs de text o paràgrafs, com aquest mateix.

  • < br >: que permet dividir el text
    en dues línies.

3. Atributs HTML

En HTML, els atributs són elements que ens porten informació extra a les diferents etiquetes del codi, fent que es modifiqui el seu comportament o aparença. Aquests, van sempre col·locats dins l'etiqueta d'obertura, i segueixen el format nom="valor"

4. Text formatat

  • < b > i < strong >: permeten aplicar negreta al text sense importància semàntica, o indicar que sí és important.

  • < i > i < em >: permeten aplicar cursiva al text sense implicar èmfasi, o indicar que sí és important.

  • < u >: permet subratllar el text.

  • < strike > o < s >: permet ratllar el text, per indicar que està borrat o ja no té rellevància.

  • < sup >: Superíndex, emprat per elevar el text.

  • < sub >: Subíndex, emprat per posar el text en posició de subíndex.
